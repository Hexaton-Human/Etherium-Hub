# Etherium-Hub
Etherium-Hub, New generate Roblox hub.

## Installation
Modern UI Version: 
- Download latest version from [releases](https://github.com/Hexaton-Human/Etherium-Hub/releases)
- Extract File
- Run Setup.exe
- Select Modern UI

Console Version: 
- Download latest version from [releases](https://github.com/Hexaton-Human/Etherium-Hub/releases)
- Extract File
- Run Setup.exe
- Select Console
